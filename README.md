# POKEDEX

Tech stacks: React Js, Hooks, SASS/SCSS, Material UI, NodeJs, Express Js


## Features
* Home
* Details
    * Catch Pokemon
* My List
    * Release Pokemon
    * Rename Pokemon

# Build Setup
* Clone repository
* Open folder [backend](https://github.com/HafidzN/BE-Pokedex),  `npm install` and `node app.js`
* Open folder pokemon and `npm install` and `npm start`
